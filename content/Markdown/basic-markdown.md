# ðŸ“– Panduan Dasar Markdown

Markdown adalah bahasa markup ringan yang digunakan untuk penulisan teks yang
bisa dikonversi ke HTML.

---

## **Heading**

Gunakan `#` untuk membuat heading. Semakin banyak `#`, semakin kecil ukurannya.

```md
# Heading 1

## Heading 2

### Heading 3

#### Heading 4

##### Heading 5

###### Heading 6
```

## **Paragraph**

Gunakann blank line untuk membuat paragraph

```md
ini adalah paragraph pertama
```

---

## **Line Breaks**

Gunakan tag `<br>` untuk membuat line breaks

```md
ini adalah baris pertama <br> ini adalah baris kedua
```

ini adalah baris pertama `<br>` ini adalah baris kedua

---

## **Bold & Italic**

Guanaakan `**` untuk membuat teks **bold** dan `*` untuk membuat teks _italic_

```md
ini adalah **bold** dan _italic_
```

ini adalah **bold** dan _italic_

---

## **Blockquote**

Gunakan `>` untuk membuat blockquote

```md
> ini adalah Blockquote
>
> > Ini adalah Blockquote kedua
> >
> > > Ini adalah Blockquote ketiga
```

> ini adalah Blockquote
>
> > Ini adalah Blockquote kedua
> >
> > > Ini adalah Blockquote ketiga

---

## **Link**

gunakan `[text](url)` untuk membuat link

```md
[facebook](https://facebook.com)
```

[facebook](https://facebook.com)

---

## **Image**

gunakan `![alt text](url)` untuk menampilkan gambar

```md
![logo](https://www.google.com/images/branding/googlelogo/1x/googlelogo_color_272x92dp.png)
```

![logo](https://www.google.com/images/branding/googlelogo/1x/googlelogo_color_272x92dp.png)
